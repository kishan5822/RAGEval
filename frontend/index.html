What gets measured

About RAGEval

RAGEval is an open-source evaluation dashboard for RAG (Retrieval-Augmented Generation) pipelines. It combines two leading evaluation frameworks — RAGAS and DeepEval — into a single interface so you can benchmark your system across 8 metrics in one run.

It does not run your RAG pipeline. You bring pre-generated outputs (questions, retrieved contexts, answers, ground truths) and RAGEval scores them using an LLM judge of your choice — Gemini, Groq, or any free OpenRouter model.

Your API key is used only for the evaluation call and is never stored server-side.

How to use

Step 1 — Get a free API key

Get a Gemini key at aistudio.google.com, a Groq key at console.groq.com, or an OpenRouter key at openrouter.ai (free models available). All three providers have a free tier.

Step 2 — Prepare your dataset (JSON)

Each row needs 4 fields:

[
  {
    "question":   "What is RAG?",
    "answer":     "RAG stands for ...",
    "contexts":   ["Retrieved chunk 1", "Chunk 2"],
    "ground_truth": "RAG is a technique that ..."
  }
]

Not sure about the format? Click Try sample dataset on this page to see a working example pre-loaded.

Step 3 — Run & read results

Pick your metrics, hit Run Evaluation, and go to the Results tab. You'll see aggregate scores, bar and radar charts, and a per-sample breakdown. DeepEval rows show an expandable reason explaining why each score was given — useful for pinpointing retrieval or generation failures.
